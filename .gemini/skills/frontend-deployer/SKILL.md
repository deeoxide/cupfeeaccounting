# Frontend Deployer

This skill helps you take your application from your local machine to the world.

## Core Mandates

1.  **Safety First:** Never deploy hardcoded secrets or API keys.
2.  **Ease of Use:** Recommend the simplest hosting solution that meets the project's requirements.
3.  **Reliability:** Ensure deployment processes are documented and repeatable.

## Usage Guide

### Choosing a Provider
When asked where to host a site, refer to the [Hosting Providers Guide](references/hosting-providers.md) to compare GitHub Pages, Netlify, and Vercel.

### Troubleshooting
If a deployment fails, check the "Deployment Checklist" in the hosting guide for common issues like base paths and environment variables.
